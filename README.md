# StravigoAI
StravigoAI is an AI-powered project intelligence platform designed for business consultancy firms to manage projects, automate timesheets and billing, track team performance, and generate actionable insights from client meetings, all in a unified .NET MAUI application supporting Android and Web deployment.

# StravigoAI  
**AI-Powered Project Intelligence for Business Consultancy Firms**

StravigoAI is a smart consulting assistant designed to centralize project management, automate billing workflows, and generate actionable insights — all in one intuitive platform. Built for business consultancy teams, StravigoAI helps track time spent, manage invoices, monitor risks, and capture meeting intelligence — enabling faster decisions and improved client satisfaction.

## Features

✔ Centralized dashboard for all operations  
✔ Add and manage client projects  
✔ Timesheet logging with billable analytics  
✔ Automated invoice generation  
✔ AI-driven project insights (local AI engine)  
✔ Team members directory and assignments  
✔ Meeting notes capture with “AI Listening Mode”  
✔ Secure session handling with login & logout  
✔ User-friendly mobile + web interface
